Projects are the top-level organizational unit in Stress Pilot.

## Creating a Project

1. In the Super App, navigate to the **Projects** tab.
2. Click the **+** (plus) button.
3. Enter a project name, description, and tags.
4. Click **Create**.

## Project Details

Once a project is created, you can access its details:

- **Dashboard**: Quick stats on test runs and status.
- **Endpoints**: List of all endpoints defined for this project.
- **Flows**: List of all test flows.
- **Environments**: Environment variables (e.g., `BASE_URL`, `AUTH_TOKEN`) specific to this project.
