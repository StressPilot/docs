Stress Pilot is a comprehensive platform designed for modern software testing. It combines the power of a Spring Boot backend, a Flutter-based "Super App", and a Python-powered AI agent to provide a seamless testing experience.

## Key Features

- **Project Management**: Organize your test suites into manageable projects.
- **Endpoint Definition**: Define and test your APIs with ease.
- **Test Flows**: Create complex test scenarios with multiple steps and dependencies.
- **Plugin System**: Extend the core functionality with custom plugins (e.g., Playwright, JavaScript, ISO8583).
- **Marketplace**: Discover and install community-contributed plugins.
- **AI Agent**: Leverage AI for automated test generation and analysis.

## Core Components

- **Stress Pilot Core**: The central Java-based engine for executing tests and managing data.
- **Plugin Service**: A Rust-based microservice for plugin management and distribution.
- **Super App**: A cross-platform UI built with Flutter for project management and monitoring.
- **AI Agent**: A Python-based agent for intelligent test orchestration.
